# Calculadora de Margen 
Calculadora en Python para calcular el margen requerido y valor nominal de posiciones de trading en Forex, Oro y BTC, con interfaz gráfica en Tkinter.

## Cómo usar
1. Instalar Python 3.x
2. Ejecutar `python main.py`
3. Ingresar precio, lote, apalancamiento y tamaño base del lote
4. Presionar "Calcular Margen" para obtener resultados
